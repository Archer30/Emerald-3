========================================================================
БИБЛИОТЕКА ДИНАМИЧЕСКОЙ КОМПОНОВКИ. Общие сведения emerald
========================================================================

Эта библиотека DLL emerald создана автоматически с 
помощью мастера приложений.

   В этом файле представлена сводка содержимого всех файлов, входящих в 
   состав приложения emerald.


emerald.vcproj
   Основной файл проекта VC++, автоматически создаваемый с помощью 
   мастера приложений.
   В файле представлены сведения о версии Visual C++, использованной 
   при создании файла, а также о параметрах платформы, конфигурации и 
   проекта, заданных с помощью мастера приложений.

emerald.cpp
   Основной исходный файл библиотеки DLL.

   При создании этой библиотеки DLL экспорт символов не выполняется. 
   Поэтому при ее построении не создается файл .lib. Чтобы определить 
   проект в качестве зависимого от другого проекта, добавьте код для 
   экспорта символов из библиотеки DLL, в результате чего будет создана 
   библиотека экспорта Кроме того, можно открыть диалоговое окно «Страницы 
   свойств», затем в папке «Компоновщик» на странице «Общие свойства» 
   установить для параметра «Пропускать входную библиотеку» значение «Да».
